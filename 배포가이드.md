# 📱 분류론 힌트집 — 배포 가이드

## 🚀 GitHub Pages로 5분 배포

### 1단계: GitHub 계정 생성 (이미 있으면 건너뜀)
https://github.com 에서 무료 가입

### 2단계: 새 저장소(repository) 만들기
1. 우측 상단 `+` → **New repository** 클릭
2. Repository name: `bunryuron` (또는 원하는 영문 이름)
3. **Public** 선택 (무료 배포 필수)
4. **Create repository** 클릭

### 3단계: 파일 업로드
1. 저장소 페이지에서 **Add file** → **Upload files** 클릭
2. 아래 파일들을 모두 드래그 업로드:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `icon-192.png`
   - `icon-512.png`
3. **Commit changes** 클릭

### 4단계: GitHub Pages 활성화
1. 저장소 상단 **Settings** 탭 클릭
2. 왼쪽 메뉴 **Pages** 클릭
3. Source: **Deploy from a branch**
4. Branch: **main** → **/(root)** 선택
5. **Save** 클릭

### 5단계: URL 확인 (1~2분 후)
```
https://[내 GitHub 아이디].github.io/bunryuron/
```
예시: https://mingstudy.github.io/bunryuron/

## 📲 폰 홈화면에 앱 추가

### Android (Chrome)
1. 위 URL을 Chrome으로 접속
2. 주소창 오른쪽 점 3개 메뉴 → **홈 화면에 추가**
3. 앱 이름 확인 → **추가** 탭

### iPhone (Safari)
1. Safari로 접속 (Chrome 아님!)
2. 하단 공유 버튼(□↑) 탭
3. **홈 화면에 추가** 탭
4. 이름 확인 → **추가** 탭

✅ 바탕화면에 📚 앱 아이콘 생성됨
✅ 오프라인에서도 작동
✅ 링크 공유로 다른 사람도 접근 가능

## 🔗 링크 공유
배포 후 URL을 카카오톡, 문자 등으로 공유하면
상대방도 바로 접근 + 홈화면 추가 가능!
